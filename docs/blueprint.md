# **App Name**: EnrollEase

## Core Features:

- Course Catalog Display: Display available courses with relevant details like title, description, schedule, and instructor. Tailwind CSS will be used to build the display components.
- User Authentication: Allow users to create accounts and log in securely. Use local storage or session storage to manage user sessions.
- Enrollment Form: Provide a dynamic form for users to select courses and fill in necessary enrollment information. Leverage Tailwind CSS for styling form elements.
- Enrollment Confirmation: Display a confirmation message or page upon successful enrollment. The confirmation screen is designed using Tailwind CSS.
- Responsive Design: Ensure the enrollment system is fully responsive across different devices (desktop, tablet, mobile) using Tailwind CSS's responsive utilities.

## Style Guidelines:

- Primary color: Deep Indigo (#4F46E5) to convey trust and reliability.
- Background color: Light Gray (#F9FAFB) for a clean and modern feel.
- Accent color: Teal (#14B8A6) for highlighting important actions and information.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Utilize a grid-based layout with Tailwind's grid system for consistent spacing and alignment.
- Use simple, outline-style icons from a library like Heroicons to represent different course categories and actions.
- Incorporate subtle transition animations for button hovers and page transitions using Tailwind's transition utilities.
